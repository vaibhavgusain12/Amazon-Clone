# Amazon Clone

This is a front-end Amazon Clone project built using only HTML and CSS.  
The project focuses on recreating the visual design and layout of the Amazon homepage for learning and practice purposes.

## Technologies Used
- HTML5
- CSS3

## Features
- Amazon-style homepage layout
- Navigation bar
- Product sections
- Footer design
- Clean and responsive UI (basic)

## Purpose
This project was created to improve HTML and CSS skills and understand real-world website layouts.
